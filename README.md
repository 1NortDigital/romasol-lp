# Romasol Engenharia — Landing Page

Landing page de captação para a Romasol Engenharia (Uberlândia · Triângulo Mineiro).
Especialistas em energia solar fotovoltaica residencial, comercial, empresarial e rural.

## Stack
- HTML/CSS/JS estático single-file
- Sem build, sem dependências
- Deploy via GitHub Pages

## Estrutura
```
romasol-lp/
├── index.html
├── assets/
│   ├── logo-romasol.png
│   └── favicon.png
└── img/
    ├── homem-preocupado-conta.jpg
    ├── 1.png  2.png  3.png  4.png   (galeria de projetos)
    └── ...
```

## Configuração
Edite as constantes no `<script>` do `index.html`:
- `WEBHOOK` — URL do n8n/Zapier para envio de leads
- `WHATSAPP_NUMBER` — número de WhatsApp (default: `553497783777`)
- `REDIRECT_OBRIGADO` — URL da página de obrigado pós-conversão
